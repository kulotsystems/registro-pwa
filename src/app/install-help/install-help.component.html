How to Install

No wifi? No problem.

Follow the steps to download the offline app!!

Instructions will appear here. Choose a browser.

Chrome Browser

Step 1 Make sure Chrome has a modern version. If unsure, update your Chrome Browser.
Step 2

Press the menu button
Step 3

Click the option Install app

NOTE If you can't see the option, update your browser

Samsung Internet

Step 1 Make sure Samsung internet has a modern version. If unsure, update your Samsung Internet App.
Step 2

Press the download app button in the app header
